# Weekly Meals + Grocery Lists

Rotating **4-week** plan — different meals every week, same macros every day (~3,300 cal · ~155 g protein). Cycles every 4 weeks. The dashboard auto-shows the current week (Plan → Meals). Batch-cook the repeated lunches/dinners (marked "batch") on Sun + Wed.
